WaveAutomate Software Release Notes

Rev 4.2                                    2/2/09 

 

Release 4.2   2/2/09

This release is compatible with the following software packages:

            WaveTest 3.5.1 or later

            WaveApps 4.1 or later

            WiMix 4.1 or later

New Features

  • Adds user settable pass/fail criteria for all test types. Setting of the acceptance criteria is optional and turned off by default.
  • Adds support for exporting results to a database. Example code provided supports MySQL. Contact Veriwave Technical Support for details.  

Fixed Issues and Improvements

  •  

Know Limitations

  • wmlToConfig.py and wmxToConfig.py are not fully functional. Conversion of some parameter may not occur correctly.

 

Release 4.0.0   7/25/08

This release is compatible with the following software packages:

            WaveTest 3.4 or later

            WaveApps 4.0.3 or later

            WiMix 4.0.2 or later

 

NOTE: PLEASE DISREGARD THE INSTALLATION PROCEDURE DESCRIBED IN THE USER GUIDE AND

      GETTING STARTED GUIDE. BOTH LINUX AND WINDOWS WAVEAUTOMATION ARE NOW CONTAINED

      IN SIMPLE ZIP FILES. JUST UNZIP THE ARCHIVE TO A DIRECTORY OF YOUR CHOOSING.

New Features

  • Adds hooks for integrating DUTs without specific integration by Veriwave.  Call Technical Support for details.

Fixed Issues and Improvements

  • Fixes problems related to specific roaming parameters that weren’t being passed correctly to the roaming test script.
  • Fixes problems related to running roaming tests with CCKM based security schemes.

Release 2.4.3.2   2/7/2008

This release is compatible with WaveTest 2.4.x up to 2.4.3.2

New Features

  • Adds support for WiMix application. Any test that can be configured in WiMix can be automated.
